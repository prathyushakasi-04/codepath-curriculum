## How Each Concept Is Structured

Every concept (except mini projects) contains:

```
concept/
├── id                  Unique ID (C01–C52)
├── title               Concept name
├── difficulty          1–5 stars
├── prerequisite_ids    Concepts that must be mastered first
├── mastery_definition  Exact criteria to be considered "mastered"
├── analogies[]         3 real-world analogies (A1 best, fallback to A2/A3)
│   └── success_rate    Updated live based on whether kid got it after hearing this
└── questions[]         4–6 questions per concept
    ├── type            predict / fill_blank / fix_bug / explain / write
    ├── code            (optional) code snippet shown to kid
    ├── question        What the kid is asked
    ├── options         Multiple choice answers (where applicable)
    ├── answer          Correct answer
    └── explanation     Why — shown after wrong answer or on request
```

---

## Question Types

| Type | Description | When Used |
|---|---|---|
| `predict` | Show code, ask what it outputs | Early in concept learning |
| `fill_blank` | Code with one piece missing | Mid learning |
| `fix_bug` | Broken code to repair | Mid learning |
| `explain` | Pick the plain-English meaning | Tests understanding vs memorization |
| `write` | Write code from a prompt | After concept is understood |

---

## Mastery Definition

A concept is considered **mastered** when:
1. Kid answers correctly 3 times in a row across different question types
2. Retention check passed: re-tested at day 3 ✓
3. Retention check passed: re-tested at day 7 ✓

A concept is flagged as **needs review** when:
- Answered incorrectly on a retention check
- 5+ wrong answers in a single session

---

## Analogy Bank — How It Works

Each concept has 3 analogies ordered by default effectiveness:
- **A1** — Best performing analogy (most kids get it after hearing this)
- **A2** — Fallback if A1 doesn't help
- **A3** — Last resort — different angle entirely

`success_rate` is updated live: if a kid hears analogy A1 and gets the next question right, A1 gets +1. Over thousands of kids, the system learns which analogy works best per age group.
